# My Personal Linktree

Este é o meu **Linktree personalizado**, desenvolvido do zero para centralizar meus links profissionais, redes sociais e projetos de portfólio. Ao contrário de ferramentas prontas, este projeto me permitiu total controle sobre o design, performance e SEO.

 **Acesse o site vivo:** [https://gustavo-linktree.netlify.app](https://gustavo-linktree.netlify.app)

## Destaques do Projeto

* **Identidade Visual:** Implementação de Favicons personalizados e ícones para dispositivos Apple (iOS).
* **Mobile-First:** Design totalmente responsivo, otimizado para navegação via smartphones.
* **Performance:** Página estática ultra-leve, garantindo carregamento instantâneo.
* **Deploy Contínuo:** Configurado com Netlify para atualizações automáticas via GitHub.
